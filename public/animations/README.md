# Hero Animation

Place your Lottie animation JSON file here as `hero.json` to display it in the HeroSection.

## How to get Lottie animations:

1. **LottieFiles.com** - Free library of animations
   - Visit https://lottiefiles.com
   - Search for animations (e.g., "job search", "career", "recruitment")
   - Download as JSON
   - Rename to `hero.json` and place in this folder

2. **After Effects + Bodymovin**
   - Create animation in After Effects
   - Install Bodymovin plugin
   - Export as JSON

3. **Design Tools**
   - Figma with Lottie plugin
   - Other tools that support Lottie export

## File Requirements:
- File name: `hero.json`
- Format: Lottie JSON (version 5.x recommended)
- Recommended size: 400x400 to 800x800 pixels
- File size: Keep under 500KB for best performance

## Current Status:
The hero section will show a placeholder until you add a `hero.json` file here.
